# Audio Files

This directory should contain background music files for the portfolio website.

## Required Files:
- `ambient-cybersecurity.mp3` - Ambient background music (320kbps MP3 recommended)

## Notes:
- Music files are not included in the repository due to size and licensing
- You can add your own royalty-free music or use services like:
  - Free Music Archive
  - YouTube Audio Library
  - Incompetech
  - Bensound

## Recommended Music Style:
- Ambient/Electronic
- Cybersecurity/Tech themed
- Loop-friendly
- Duration: 2-5 minutes
- Format: MP3, 320kbps
- Size: < 10MB

## Usage:
Place your MP3 file in this directory and ensure it's named `ambient-cybersecurity.mp3` or update the src in index.html accordingly.
